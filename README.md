# InkLedger-java-application
InkLedger is an application for managing school supply inventory and services like printing, photocopying, and laminating. It tracks stock, issued supplies, and generates reports showing net income with borrow deductions. Designed for small businesses, it helps stay organized, reduce manual work, and make smarter financial decisions.
